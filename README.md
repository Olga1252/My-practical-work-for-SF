# Практикум по направлению исследований экологических проблем 
## На тему "Дефорестация как экологическая угроза"

### Абстракт
Данная работа была составлена в рамках практикума, где рассматривается тема изрежения леса. За основую были взяты снимки со спутника и почти готовая модель Resnet18. Исследование заключалось в том,что путем классификации было рассмотрено два типа снимков - с дефорестацией и с лесом. 

### Формулирование проблемы 
Дефорестация ( с англ. deforestation) может нести в себе довольно значительную угрозу для экологии. Во-первых, дефористация проявляется уменьшением объема растительности в лесных территориях, тем самым сокращая ареал обитания локальной фауны и флоры (здесь может возникнуть проблема с нарушением пищевой цепочки у лесного биоразнообразия, сокращение краснокнижных растений и животных, нарушение экосистемы леса в целом). В дополнении  к этому, довольно частой проблемой, сопутствующей последствиям вырубки леса , является эррозия: как почвенная, так и ветровая (здесь может возникнуть проблема "выветривания" почвенного покрова и возникновению пыльных бурь; во время обильных осадков - сель, или временное заболачивание из-за отсутствия или снижения удержания осадков почвой). Однако, мониторинг лесной местности может своевременно предотвратить чрезмерную дефорестацию путем принятия согласованных мер по предотвращению не только уменьшения процентов растительности ,но и принятию мер по снижению последствий от данной катастрофы.

### Цель: 

Исследовать на выбранном датасете ,какое изображание находится на спутниковом снимке: лес или дефорестация. 
Тип задачи - бинарная классификация изображений. 
Область - компьютерное зрение

### Методология 
В рамках рассмотрения проекта, был взят датасет для создания бинарной классификации со спутниковыми снимками. Работа осуществлялась с помощью Google Colaboratory. Модель для составления бинарной классификации является Resnet18 hugging face. Данная модель была подобрана в соотвествии с исходными техническими требованиями со стороны датасета , а именно:
- работа с невидимым количеством уровней,
- использование для бинарной классификации,
- видит общие объекты.


Это CV модель.

input: картинка со спутника (RGB)
output: класс forest = 1, deforestation =0 . 

В результате мини-исследования было выялено ,что модель научилась распознавать два состояния снимков (дефорестация / лес), у модели высокая точность валидации , при выборе номера картинки в датасете модель правильно предсказывает ее, код работает.

### Пробелы в исследовании. 
Бинарная классификация спутниковых снимков может стать неким вспомогательным инструментом для исследования проблем леса, однако для того ,чтобы оптимизировать процесс решения поставденных задач относительно решения экологической проблемы, стоило бы внедрить тип задач по сегментации, чтобы дополнительно оценивать процентное соотношение вырубленных участков леса.
